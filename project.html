Featured Project: Wallyland Java Application

Project Overview:

The Wallyland Java Application is a comprehensive system designed for a fictional theme park, Wallyland. This application serves both visitors and employees, providing features such as ticket purchasing, ticket searching, user creation and management, and interactive maps.

Key Features:

Ticket Management: Allows users to purchase and search for tickets efficiently.

User Management: Facilitates user creation and management with CRUD (Create, Read, Update, Delete) operations.

Interactive Maps: Provides a visual guide of the theme park for an enhanced visitor experience.

Technical Details:

Persistence: Utilizes Serializable to ensure data persistence for users and tickets.

Data Organization: Employs hash maps and array lists to manage data.

Development Tools: Developed using Apache NetBeans and version-controlled via GitHub.

Team Project:

This was a collaborative effort within a group, showcasing teamwork and version control practices.
